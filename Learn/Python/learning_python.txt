1、第一章我们学习了冯诺依曼机的基本组成及工作原理。
    冯·诺依曼机是一种计算机设计架构，它的核心思想是将程序和数据以二进制形式存储在计算机的内存中，使得计算机可以按照存储的指令顺序执行操作。
    这种架构包括五个主要部分：存储器、控制器、运算器、输入设备和输出设备。
    存储器用于存放程序和数据，控制器负责解析指令并控制其他部件，运算器执行数学和逻辑运算，输入设备和输出设备分别处理数据的输入和输出。
    这种设计使得计算机能够灵活地执行各种程序，而不是仅限于执行固定的操作。
