# HTML 
> HyperText Markup Language（超文本标记语言）

其文件后缀名为".html"、".htm"

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
## 其中

`<!DOCTYPE html>`
> 声明文档为HTML5文档
